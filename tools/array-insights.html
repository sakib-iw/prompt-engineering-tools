Number array insights

Paste a JSON array of numbers (model output, shuffled IDs, lottery-style lists). Get counts, min/max, sum, mean, median, monotonicity, sorted lists, duplicate indices, index↔value table, gaps in an integer range, and consecutive runs — plus a copyable plain-text report for prompts.

Leave both range fields empty to find missing integers between the array’s minimum and maximum. Set both to scan a fixed band (e.g. 1–52).

Input rules: valid JSON only — square brackets, comma-separated numbers. Missing integers use each whole number k in the chosen range and check whether k appears exactly in the array. Consecutive runs need all values to be integers.
