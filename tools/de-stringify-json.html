De-stringify JSON

Paste JSON that contains another JSON document as an escaped string (sometimes repeatedly). This tool parses until it reaches a real object or array, then pretty-prints the result.

How it works: JSON.parse is applied to your text, then again on the result while the value is still a string (with a safety cap on depth). If the final value is not a string, it is formatted with JSON.stringify(..., null, 2).
